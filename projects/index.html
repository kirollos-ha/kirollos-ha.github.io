Jelly

jelly-less-proto

Jelly (java embeddable lisp interpreter) was an learning project (later turned exam project first, and then thesis) of mine aimed at developing a small scheme implementation. It features most of what you'd expect from a scripting language and from a lisp, such as first order functions, higher order functions, lexical bindings, and the like.
It also has a Java foreign function interface, (althoug it's implemented via reflection, so not the most performant ffi out there, but more than enough for a learning project) and a module system based on a subset of r7rs, on top of which jelly also implements a small standard library

Please note I came up with the name Jelly long before learning there were, it seems other languages named jelly, and by the time I realized it it was too late.


CIRA

cira

CIRA (short for "Coreutils In Rust, Again") is a small set of unix coreutil clones written as an excercise in rust.
So far only cat an wc have been implemented.
